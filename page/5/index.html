function add(x,y) {return x + y};
but 我也也会有这种抽象
function add3
function add5
function add7
所以我们会做一个面向横切面的抽象 eg 
function addWrap(y) {
  return function(x) {
    
  }
} 
function add3 = addWrap(3);
function add5 = addWrap(5);
function add7 = addWrap(7);
So，真实只是申明了一个Function，其他都是调用addWrap
换用到C#逻辑也是通用。C# 类型中也需要这样程度的抽象，这个在C#里被称为 泛型。
